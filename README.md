# Expense Tracker App 
A simple and responsive expense tracker app that helps users manage income and expenses efficiently.
## Live Demo 
https://aujlaashmeen.github.io/Expense-Tracker/
## Tech Stack
- HTML
- CSS
- JavaScript
## Features 
- Add income and expense transactions
- Automatically calculates balance
- Delete transactions
- Data persistence using localStorage
- clean and responsive UI
